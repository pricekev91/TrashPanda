# scripts/

This directory holds automation and utility scripts for TrashPandaOmega.

Examples of scripts to add here:

- `bootstrap.sh` — set up a fresh development environment
- `lint.sh` — run linters (e.g., `ruff`, `mypy`)
- `test.sh` — run the full test suite
- `release.sh` — bump version and update CHANGELOG

> Scripts should be self-contained, well-commented, and idempotent where possible.
